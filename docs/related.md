## Related Projects

Matchzy Enhanced is part of a larger ecosystem of CS2 tournament automation tools. Here are the related projects:

### CS2 Server Manager

**[CS2 Server Manager](https://csm.sivert.io/)** – Multi-server CS2 deployment and management platform.

CS2 Server Manager helps you deploy, configure, and manage multiple CS2 servers. It's designed to work seamlessly with Matchzy Enhanced for tournament automation.

### MatchZy Auto Tournament

**[MatchZy Auto Tournament](https://mat.sivert.io)** – Web UI and API for automated CS2 tournaments.

MatchZy Auto Tournament orchestrates tournament brackets, match scheduling, and team management. It integrates with Matchzy Enhanced to provide a complete tournament automation solution.

## How They Work Together

- **CS2 Server Manager** handles server deployment and infrastructure
- **Matchzy Enhanced** (this project) manages in-game match execution
- **MatchZy Auto Tournament** orchestrates tournament brackets and scheduling

Together, these tools provide a complete solution for running automated CS2 tournaments at scale.
